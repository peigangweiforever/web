# CSS3图片边框效果
## 介绍
> CSS3可以通过设置图片来做边框，可以使网页变得更加漂亮。
## 属性
+ [border-image](http://css.doyoe.com/properties/border/border-image.htm)
+ [border-image-source](http://css.doyoe.com/properties/border/border-image-source.htm)
+ [border-image-width](http://css.doyoe.com/properties/border/border-image-width.htm)
+ [border-image-slice](http://css.doyoe.com/properties/border/border-image-slice.htm)
+ [border-image-outset](http://css.doyoe.com/properties/border/border-image-outset.htm)
+ [border-image-repeat](http://css.doyoe.com/properties/border/border-image-repeat.htm)
